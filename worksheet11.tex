\begin{questions}

\question Suppose that g is a function from A to B and f is a function from B to C. Prove each of these statements

\begin{parts}
\part
If f◦g is onto, then f must also be onto
\vspace{9in}

\part
If f◦g is one-to-one, then g must also be one-to-one
\vspace{9in}

\part
If f◦g is a bijection, then g is onto if and only if f is
one-to-one

\vspace{9in}
\end{parts}
\question
Let F : $X \rightarrow Y$ be a function. Show $F^-1$ is a function from Y to X if and only if F is a 1-1 correspondence.
\vspace{9in}
